
Setup Rapidissimo in PHPStorm:
# Crea il progetto

File → New Project → PHP Empty Project
Scegli una cartella e crea

# Crea i file PHP

Tasto destro sulla cartella → New → PHP File
Crea form.php (o index.html + elabora.php)
Incolla il codice dell'esempio

# Avvia il server built-in

## Modo A - Click destro sul file:

Tasto destro su form.php
Seleziona "Open in Browser" o "Run 'form.php'"
PHPStorm avvia automaticamente il server built-in PHP! 

## Modo B - Configurazione manuale (una volta sola):

Run → Edit Configurations
Click + → PHP Built-in Web Server
Configura:

Host: localhost
Port: 8000
Document root: lascia la root del progetto
Router script: lascia vuoto per ora


Apply e OK
Clicca il pulsante verde Run ▶️ nella toolbar
